# Web Performance Optimization

Performance is crucial for user experience and SEO. Here are five techniques I've found effective in my projects.

## 1. Image Optimization

Always optimize your images before deployment. Here's a simple Node.js script I use:

```javascript
const sharp = require('sharp');

async function optimizeImage(input, output) {
  await sharp(input)
    .resize(800, null, {
      withoutEnlargement: true,
    })
    .webp({ quality: 80 })
    .toFile(output);
}
```

## 2. Code Splitting

React makes code splitting simple with dynamic imports:

```javascript
const MyComponent = React.lazy(() => import('./MyComponent'));
```

## 3. Caching Strategies

Implement proper caching headers:

```typescript
export async function getServerSideProps({ res }) {
  res.setHeader(
    'Cache-Control',
    'public, s-maxage=10, stale-while-revalidate=59'
  );

  return {
    props: {},
  };
}
```

## Performance Metrics to Monitor

- First Contentful Paint (FCP)
- Largest Contentful Paint (LCP)
- Time to Interactive (TTI)
- Cumulative Layout Shift (CLS)
